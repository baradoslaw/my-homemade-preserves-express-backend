<!-- GETTING STARTED -->
## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/baradoslaw/my-homemade-preserves-express-backend.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the server development
```sh
   npm start
   ```
